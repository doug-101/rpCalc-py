rpCalc Downloads

Linux

On a Linux system that has the libraries noted on the requirements page, all that is needed is the source code (with an install script), included in rpcalc-0.8.2.tar.gz (48 KB).

But please read about the installation and use of rpCalc. This information is available on this site and in the ReadMe file inside the above tar file.

Windows

Two windows installers are available. Use rpcalc-0.8.2-install-all.exe (11 MB) to install for all users (administrator permissions are required). Use rpcalc-0.8.2-install-user.exe (11 MB) to install for a single user (administrator rights are not required). The source code for the libraries used to build the Windows binaries is available upon request.

But please read about the installation and use of rpCalc. This information is available on this site and in the ReadMe file included in the installer.
